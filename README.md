


# Inventory Reservation System

A frontend-based Inventory Reservation System built using HTML, CSS, and JavaScript.

## Features

- Product Listing
- Temporary Inventory Reservation
- Countdown Timer
- Auto Expiry
- Reservation History
- Concurrency Handling
- LocalStorage Persistence
- Toast Notifications
- Responsive UI
